# 2.4 — Media & Content Notes

Content-packaging notes for the lesson **2.4 — Reading Real Instrument Files**.
These are production notes only; the technical content lives in the notebook.

## Working title

*Getting Your Instrument Data Into Python (Without Losing Your Mind)*

## YouTube packaging

- **Thumbnail angle:** split screen — a wall of raw header junk on the left, a
  clean spectrum plot on the right, with "WHY WON'T MY CSV LOAD?" across the top.
- **First 15 seconds (the hook):** show File B loading silently wrong (numbers as
  text), then the one-line fix. Lead with the *silent* failure — it's the one
  that scares people.

## Viewer challenge (call to action)

The `exports/` folder is yours. Write a *fifth* file with a new combination —
say, semicolons *and* a metadata header *and* a footer — and confirm
`load_spectrum` still recovers it. If it doesn't, you've found the next quirk
worth handling.
